# Taskora - Flutter App

## GitHub pe daalne ke steps:

### Step 1 - GitHub account banao
1. github.com kholo
2. "Sign Up" karo (free)
3. Username, email, password dalo

### Step 2 - New Repository banao
1. github.com/new pe jao
2. Repository name: `taskora`
3. Public select karo
4. "Create repository" click karo

### Step 3 - Files upload karo
1. "uploading an existing file" click karo
2. Ye saari files upload karo:
   - `pubspec.yaml`
   - `lib/main.dart`
   - `android/app/src/main/AndroidManifest.xml`
3. "Commit changes" click karo

### Step 4 - Codemagic pe build karo
1. codemagic.io kholo
2. GitHub se login karo
3. "Add application" click karo
4. Apna `taskora` repo select karo
5. Flutter select karo
6. "Start your first build" click karo
7. 10-15 min mein APK ready!
8. Download karo → Phone pe install karo ✅
